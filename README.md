# git-aliases
Collation of my own git aliases.

## List of aliases
```bash
git config alias.a "add ."
git config alias.br branch
git config alias.c "commit"
git config alias.ca "commit --amend"
git config alias.cm "commit -m"
git config alias.co checkout
git config alias.cob "checkout -b"
git config alias.d diff
git config alias.ds "diff --staged"
git config alias.dt difftool
git config alias.fet fetch
git config alias.mt mergetool
git config alias.pf "push -f"
git config alias.rb rebase
git config alias.rbi "rebase -i"
git config alias.rbc "rebase --continue"
git config alias.st status
git config alias.tree "log --all --graph --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan) %an%C(reset) | %C(bold green)%aD%C(bold yellow) (%ar)%C(reset) | %C(dim white)%d%C(reset)%n           %s'"
```
